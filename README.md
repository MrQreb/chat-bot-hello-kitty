# Chatbot

## Requirements

- Nodejs 22.x 
- Prerequisistes of ***tauri*** ***https://v1.tauri.app/v1/guides/getting-started/prerequisites/***
- Install npm depedencies ```yarn install```
- Get gemini api key ***https://ai.google.dev/gemini-api/docs/api-key***

## Run dev server 

Run in web browser
```bash
yarn dev
```

Run in desktop app
```bash
yarn tauri dev
```

## Create build
```bash
yarn tauri build
```
